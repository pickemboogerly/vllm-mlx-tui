# go-wave (DEPRECATED)

> [!WARNING]
> This folder contains the deprecated Go (Bubble Tea) implementation of vLLM-wave.
> It has been officially retired as a **failed experiment** due to architecture, concurrency, and stability challenges related to the intricate polling of background processes within the TUI event loop.

Please use the Python implementation (`python-wave`) which is now the canonical, supported version of this project.

## How to proceed
Do not develop further in this module. Refer to `../PRD.md` for current project documentation.
